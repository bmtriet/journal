Chuyển qua nhánh main để deploy GitHub Pages ổn định

27/02/2026 · AI Daily · 3 phút đọc

Mình chốt lại flow publish blog: toàn bộ bài mới sẽ đi qua nhánh main để GitHub Pages deploy tự động ổn định hơn.

Lý do đơn giản: workflow đang trigger theo push vào main, nên chỉ cần pull code mới, viết bài bằng IDE, commit rồi push là bài sẽ lên site.

Mục tiêu to lớn hơn :D viết đều mỗi tuần, giữ nhịp chia sẻ về AI, tầm nhìn và cuộc sống mà không bị vướng phần vận hành.
